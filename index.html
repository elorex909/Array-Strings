Collections of Data

C++ Arrays

Arrays Explained

The Concept

An array is a fixed-size collection of elements of the same type stored in contiguous memory.

Think of it as a row of boxes, each with an index starting from 0.

int arr[5] = {10, 20, 30, 40, 50};

// Accessing
cout << arr[0]; // 10
cout << arr[4]; // 50

// Warning: arr[5] is out of bounds!

Garbage Values

Initialization Rules

Global Arrays: Automatically initialized to 0.

Local Arrays: Contain "Garbage Values" (random numbers) unless initialized manually.

int globalArr[5]; // All 0

int main() {
    int localArr[5]; 
    // Contains random junk!
    
    int cleanArr[5] = {0}; 
    // Force initialize to 0
}

Sorting & Searching

#include <algorithm>

C++ provides powerful global functions for arrays:

  • sort(arr, arr+n): Sorts ascending.
  • reverse(arr, arr+n): Flips order.
  • *max_element(arr, arr+n): Finds max.
  • count(arr, arr+n, val): Counts occurrences.
#include <algorithm>
int a[5] = {5, 1, 4, 2, 8};

sort(a, a+5); 
// a is {1, 2, 4, 5, 8}

cout << *max_element(a, a+5);
// Output: 8

Descending Sort

Sorting Backwards

By default, sort() is ascending (Low to High).

To sort descending (High to Low), add a third argument: greater<type>().

int arr[] = {1, 5, 3};

// Sort Descending
sort(arr, arr+3, greater<int>());

// arr is now {5, 3, 1}

Working with Text

C++ Strings

Strings Explained

Dynamic Text

std::string is a dynamic array of characters. It handles memory automatically and allows easy concatenation.

#include <string>

string name = "Ahmed";
string msg = "Hello " + name;

cout << msg.size(); // Length
cout << msg[0];     // 'H'

String Operations

Class Methods

  • .push_back('c'): Add char to end.
  • .pop_back(): Remove last char.
  • .find("sub"): Find index of substring.
  • .substr(idx, len): Extract part of string.
string s = "Code";
s.push_back('s'); // "Codes"

int idx = s.find("de"); // 2

string sub = s.substr(0, 2); 
// "Co"

Iterators: begin() & end()

Sorting Strings

To use sort() or reverse() with strings, we use Iterators instead of simple pointers.

  • s.begin(): Points to start.
  • s.end(): Points to after the last char.
string s = "cba";

// Sort Alphabetically
sort(s.begin(), s.end());
// s becomes "abc"

// Reverse String
reverse(s.begin(), s.end());
// s becomes "cba" again

Conversions

Number â†” String

Very common in CP problems.

  • to_string(num): Converts int/double to string.
  • stoi(str): Converts string to integer.
int x = 123;
string s = to_string(x);
// s is "123"

string numStr = "50";
int y = stoi(numStr);
// y is 50

Character Checks

<cctype> Library

Check or modify individual characters:

  • isdigit(c): Is it 0-9?
  • isalpha(c): Is it A-Z or a-z?
  • toupper(c): Convert to Uppercase.
  • tolower(c): Convert to Lowercase.
char c = 'a';

if(isalpha(c)) {
    c = toupper(c);
}
// c is now 'A'

Input Pitfalls

cin vs getline

Mixing cin and getline causes issues. cin leaves a newline in the buffer, which getline reads immediately.

Solution: Use cin.ignore().

int id;
string name;

cin >> id;
// Ignore the newline left by cin
cin.ignore(); 

getline(cin, name);

Problem A: Array Sum (Easy)

Description

Given an array of 5 integers, calculate the total sum using a loop.

Task: Read 5 numbers and print sum.

Input10 20 30 40 50
Output150

Solution A

Code

int main() {
    int arr[5], sum = 0;
    for(int i=0; i<5; i++) {
        cin >> arr[i];
        sum += arr[i];
    }
    cout << sum;
}

Problem H: Palindrome Check (Hard)

Description

Check if a string reads the same forwards and backwards.

Task: Create a reversed copy and compare.

Inputmadam
OutputYES

Solution H

Code

Leveraging reverse() logic.

#include <algorithm>
int main() {
    string s; cin >> s;
    string rev = s;
    
    reverse(rev.begin(), rev.end());
    
    if(s == rev) cout << "YES";
    else cout << "NO";
}
